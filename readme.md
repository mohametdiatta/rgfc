# rgfc

> A react CLI for generating components

## Install

```bash
$ npm install --global rgfc
# OR
$ yarn add --global rgfc

```

## CLI

```
$ rgfc --help

  Usage
	  $ rgfc

	Options
		--component component-name
		-c component-name

	Examples
	  $ rgfc -c component-name
     $ rgfc -c path/component-name
```
